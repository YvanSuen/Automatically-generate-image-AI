**********************代码说明**********************
本工程下各文件的功能如下：
images：训练所用的图片（本项目只上传了20%的训练图片，训练图片的数量包括图片的内容可自行选择）
generate：训练1000次后的模型输出样例（训练epoch更多的话效果会更真实）
network.py：DCGAN 深层卷积生成对抗网络
train.py：训练 DCGAN
generate.py：用 DCGAN 的生成器模型 和 训练得到的生成器参数文件 来生成图片
